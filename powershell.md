# Customizing PowerShell

> Наприкінці статті ви знайдете посилання на приклади побудови різноманітних функцій від роботи з Git до додавання команд Linux.

PowerShell — це потужний кросплатформний фреймворк для автоматизації завдань та керування конфігурацією, розроблений Microsoft. Він поєднує в собі оболонку командного рядка з мовою сценаріїв та фреймворком для обробки командлетів.

## Власні функції (командлети)

PowerShell дозволяє створювати і використовувати власні функції/командлети (надалі - функції).
Для початку необхідно створити файл профілю користувача в PowerShell.
Для цього за шляхом (`user_name` вам потрібно замінити на вашого користувача): `C:\Users\user_name\OneDrive - Raiffeisen Bank International Group\Documents\` створіть теку `PowerShell` і в ній створіть файл `Microsoft.PowerShell_profile.ps1`.

> **Всі функції надалі потрібно створювати саме в цьому фалі.**

Після створення файлу профіля, ви зможете викликати його на редагування або перезавантаження за допомогою змінної `$PROFILE`

Редагування:

```shell
notepad $PROFILE
```

Перезавантаження:

```shell
. $PROFILE
```

Перезавантаження потрібне, щоб вступили в силу зміни, яки ви внесли в файл профіля. Команда перезавантаження діє лише на поточний сеанс PowerShell.

Перезавантаження профіля потрібно робити для кожної відкритої сесії PowerShell після внесення змін у файл профіля.

---

> **Примітка:** Якщо ви використовуєте VS Code, ви можете відкрити файл профіля безпосередньо в редакторі за допомогою команди:

```shell
code $PROFILE
```

## Приклад профілю

В [файлі-прикладі](examples/PowerShell/profile.ps1) наведено приклад профілю PowerShell з кількома корисними функціями, які можна використовувати для покращення вашого робочого процесу.
Ці функції включають:
- Функції для роботи з GIT
- Функції для навігації по файловій системі в стилі Linux
- Функції створення та імпорту валідного сертифіката для HTTPS localhost
- Функції для роботи з MerchantDesc/Binder
- Функції перегляду помилок

> В профілі доступна функція `help`, яка виводить список усіх > доступних функцій з коротким описом їх призначення.


Ви можете скопіювати цей файл у свій профіль PowerShell або налаштувати його відповідно до ваших потреб.
